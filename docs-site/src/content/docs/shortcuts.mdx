---
title: Keyboard Shortcuts
description: Complete reference of keyboard shortcuts available in CrossTide.
---

CrossTide provides keyboard shortcuts for fast navigation and common actions.
Press **?** anywhere to open the shortcuts overlay.

## Navigation

| Key | Action |
|-----|--------|
| `W` | Open Watchlist |
| `C` | Open Chart |
| `S` | Open Screener |
| `A` | Open Alerts |
| `P` | Open Portfolio |
| `B` | Open Backtest |
| `H` | Open Heatmap |
| `,` | Open Settings |
| `?` | Show shortcuts overlay |

## Actions

| Key | Action |
|-----|--------|
| `/` | Focus search / add ticker |
| `Esc` | Close modal / cancel action |
| `Ctrl+E` | Export data (CSV) |
| `Ctrl+K` | Open command palette |
| `R` | Refresh current card data |
| `F` | Toggle fullscreen chart |

## Chart-Specific

| Key | Action |
|-----|--------|
| `1` – `6` | Switch timeframe (1D / 5D / 1M / 3M / 1Y / 5Y) |
| `D` | Toggle drawing tools panel |
| `←` / `→` | Pan chart left / right |
| `+` / `-` | Zoom in / out |

## Accessibility

All shortcuts can be remapped in **Settings → Keyboard Shortcuts** (coming soon).
Screen reader users can navigate via standard landmark regions and ARIA labels.
